How do I see the differences between HTML and CSS?

  1. What is the difference between the usage of HTML and CSS?

    HTML file contains all the content on the page, CSS defines how the content will look.

  5. What is the difference between semantic and non-semantic elements?

    Semantic element defines its content, as do header, section, article... Non-semantic element does not indicate its content, such as span and div.

  6. Why do we prefer using semantic elements?

    Using semantic tags is generally preferred over a div-based structure because it adds clarity and makes code more mantainable.

  7. Do all browsers support all of the newest HTML / CSS elements?

    Not always, it is advisable to verify the compatibility of browser and intended element.
